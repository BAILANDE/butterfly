隐私政策（中文）

最后更新日期：2026-03-12

欢迎访问本博客（以下简称 “本网站”）。作者非常重视访问者的隐私与数据安全。本隐私政策旨在说明当您访问或使用本网站时，可能涉及的信息收集、使用、存储与保护方式。

在访问或使用本网站前，请仔细阅读本隐私政策。
继续访问或使用本网站即表示您已阅读、理解并同意本政策内容。

一、网站性质说明

本网站为 个人博客及静态网站，主要用于分享作者的：

  • 技术文章
  • 学习笔记
  • 个人观点
  • 生活记录

本网站基于以下技术构建：

  • Hexo 静态博客框架
  • Butterfly 主题

本网站 不提供以下功能：

  • 用户注册系统
  • 用户账号体系
  • 在线交易或支付功能
  • 用户信息提交表单

因此，本网站 不会主动收集访客的敏感个人身份信息（例如身份证号、住址等）。

但在部分情况下，第三方服务或评论系统可能会收集必要的信息。

二、可能收集的技术信息

当您访问本网站时，服务器或相关网络基础设施可能自动记录一些 非个人识别信息（Non-Personal Information），包括但不限于：

  • IP 地址
  • 浏览器类型及版本
  • 操作系统类型
  • 设备类型（电脑 / 手机 / 平板）
  • 访问时间
  • 来源页面（Referrer）
  • 访问页面路径
  • 访问频率
  • 页面停留时间

这些信息通常由以下系统自动记录：

  • 网站托管服务
  • CDN 网络
  • 网络服务器日志
  • 网站统计工具

这些数据 仅用于技术分析、访问统计和安全维护，不会用于识别特定个人身份。

三、第三方服务

为了保证网站正常运行，本网站可能依赖部分 第三方服务。这些服务可能根据其自身隐私政策收集必要的技术信息。

1. 网站托管服务

本网站可能托管于以下平台：

  • GitHub Pages
  • Vercel

这些平台可能会记录访问日志和网络请求信息。
相关数据处理方式以对应平台的隐私政策为准。

2. 内容分发网络（CDN）

为了提高网站访问速度和稳定性，本网站可能使用 CDN（内容分发网络） 缓存静态资源，例如：

  • JavaScript
  • CSS
  • 图片
  • 字体
  • 图标资源

CDN 服务在提供资源时可能记录基本网络信息，例如：

  • IP 地址
  • 请求时间
  • 请求资源

3. 第三方插件与资源

本网站基于 Hexo + Butterfly 主题 构建，并可能使用部分第三方插件或资源，例如：

  • JavaScript 开源库
  • 图标库
  • 字体服务
  • 音乐播放器
  • 页面动画效果插件
  • 页面统计工具

这些资源可能来自公共 CDN 或第三方服务器。

当浏览器加载这些资源时，相关服务器可能会记录访问请求信息。

四、评论系统

为了提供互动功能，本网站使用 第三方评论系统。

当访客发表评论时，评论系统可能会收集以下信息：

  • 昵称
  • 邮箱地址
  • 评论内容
  • IP 地址
  • 浏览器信息
  • 评论时间

这些信息可能用于：

  • 显示评论内容
  • 防止垃圾评论
  • 评论回复通知
  • 评论系统运行维护

评论数据通常 存储在评论系统服务提供商的服务器上，而非本网站服务器。

如果您不希望提供上述信息，可以选择 不使用评论功能。

五、Cookie 与本地存储

本网站 不会主动设置用于跟踪用户行为的 Cookie。

但在以下情况下，浏览器可能产生 Cookie 或本地存储：

  • CDN 缓存机制
  • 评论系统
  • 网站统计工具
  • 第三方插件
  • 浏览器缓存

Cookie 可能用于：

  • 提高网站访问速度
  • 保存用户偏好
  • 统计访问数据
  • 支持评论系统功能

您可以通过浏览器设置：

  • 禁用 Cookie
  • 删除 Cookie
  • 阻止第三方 Cookie

禁用 Cookie 可能会影响部分网站功能。

六、信息使用目的

本网站及相关第三方服务产生的技术数据仅用于以下目的：

  • 维护网站安全
  • 防止恶意访问或网络攻击
  • 分析访问趋势
  • 优化网站结构
  • 提升页面加载速度
  • 改善用户体验

作者 不会进行以下行为：

  • 出售访客信息
  • 出租访客信息
  • 交换访客信息
  • 将访客信息用于商业营销

七、信息共享与披露

本网站不会主动向第三方出售或披露访客信息，除非出现以下情况：

  • 法律法规要求
  • 司法机关依法要求
  • 为保护网站安全
  • 为防止网络攻击或滥用行为

在上述情况下，可能会披露必要的技术信息。

八、外部链接

本网站可能包含指向其他网站的链接，例如：

  • 技术文档
  • 开源项目
  • 参考资料
  • 外部博客

这些网站拥有各自独立的隐私政策。

作者对第三方网站的：

  • 数据收集行为
  • 隐私政策
  • 内容安全

不承担任何责任。

建议访问者在访问外部网站前阅读其隐私政策。

九、数据安全

作者会尽合理努力维护网站安全，例如：

  • 使用可靠的托管平台
  • 避免收集不必要数据
  • 定期更新网站程序
  • 限制服务器访问权限

但由于互联网的开放性与复杂性，作者 无法保证数据传输的绝对安全性。

访问者在使用互联网服务时应理解并承担相关风险。

十、未成年人隐私保护

本网站内容面向一般公众，并未专门针对未成年人设计。

若未成年人访问本网站，建议在 监护人指导下使用互联网服务。

十一、用户数据权利

在适用法律允许的范围内，访问者可能拥有以下权利：

  • 查询相关数据
  • 请求删除评论数据
  • 请求更正错误信息

如果您希望删除您的评论或相关数据，可以通过下方联系方式联系作者。

十二、隐私政策更新

作者可能根据以下情况更新本隐私政策：

  • 网站功能变化
  • 第三方服务变更
  • 法律法规更新

更新后的隐私政策将在本页面发布，并标注最新更新时间。

继续使用本网站即表示您接受更新后的隐私政策。

十三、联系方式

如果您对本隐私政策有任何疑问、意见或建议，可以通过以下方式联系作者：

电子邮箱

moxijinlvyi@email.com

简要总结

本网站：

  • 不提供用户账户系统
  • 不收集敏感个人身份信息
  • 可能记录匿名访问数据
  • 使用第三方评论系统
  • 可能使用第三方插件与 CDN

访问本网站即表示您同意本隐私政策。

Privacy Policy

Welcome to this blog (hereinafter referred to as “this Website”).
The author values the privacy and data security of all visitors. This Privacy Policy explains how information may be collected, used, stored, and protected when you visit or use this Website.

Please read this Privacy Policy carefully before using this Website.

This Website is a personal blog and static website primarily used to share:

  • Technical articles
  • Study notes
  • Personal opinions
  • Life records

This Website is built using:

  • Hexo static blog framework
  • Butterfly theme

This Website does NOT provide the following services:

  • User registration system
  • User account system
  • Online transactions or payments
  • User information submission forms

Therefore, this Website does not actively collect sensitive personal identification information, such as identification numbers or home addresses.

However, some third-party services or comment systems used on the Website may collect necessary information.

When you visit this Website, servers or related network infrastructure may automatically record certain non-personally identifiable information, including but not limited to:

  • IP address
  • Browser type and version
  • Operating system
  • Device type (computer / phone / tablet)
  • Access time
  • Referrer page
  • Page access path
  • Access frequency
  • Page dwell time

These records may be automatically generated by:

  • Website hosting services
  • CDN networks
  • Web server logs
  • Website analytics tools

Such data is used only for technical analysis, traffic statistics, and security maintenance, and is not used to identify specific individuals.

3. Third-Party Services

To ensure the proper operation of this Website, certain third-party services may be used. These services may collect necessary technical information according to their own privacy policies.

3.1 Website Hosting Services

This Website may be hosted on platforms such as:

  • GitHub Pages
  • Vercel

These platforms may record access logs and network request information.

Please refer to the respective platforms’ privacy policies for details on how they handle data.

3.2 Content Delivery Network (CDN)

To improve performance and loading speed, this Website may use a Content Delivery Network (CDN) to distribute static resources such as:

  • JavaScript files
  • CSS files
  • Images
  • Fonts
  • Icon resources

CDN providers may record basic network request information, such as:

  • IP address
  • Request time
  • Requested resources

3.3 Third-Party Plugins and Resources

This Website is built using Hexo + Butterfly theme, and may include third-party plugins or resources, such as:

  • JavaScript open-source libraries
  • Icon libraries
  • Font services
  • Music player components
  • Animation or visual effect plugins
  • Analytics scripts

These resources may be delivered via public CDNs or third-party servers. When your browser loads these resources, the corresponding servers may record basic request information.

4. Comment System

To provide interactive features, this Website uses a third-party comment system.

When visitors post comments, the comment system may collect certain information, including:

  • Nickname
  • Email address
  • Comment content
  • IP address
  • Browser information
  • Comment timestamp

This information may be used for purposes such as:

  • Displaying comments
  • Preventing spam
  • Sending reply notifications
  • Maintaining comment system functionality

Comment data is usually stored on the servers of the comment system provider, rather than on this Website’s server.

If you do not wish to provide such information, you may choose not to use the comment function.

5. Cookies and Local Storage

This Website does not intentionally set tracking cookies.

However, cookies or local storage may be created in certain cases, including:

  • CDN caching mechanisms
  • Comment systems
  • Website analytics tools
  • Third-party plugins
  • Browser caching

Cookies may be used to:

  • Improve website performance
  • Save user preferences
  • Analyze traffic statistics
  • Support comment system functionality

You may manage cookies through your browser settings by:

  • Disabling cookies
  • Deleting cookies
  • Blocking third-party cookies

Disabling cookies may affect certain website functions.

6. Purpose of Data Usage

Technical data generated by this Website or related third-party services may be used for the following purposes:

  • Maintaining website security
  • Preventing malicious access or cyber attacks
  • Analyzing traffic patterns
  • Improving website structure and content
  • Enhancing page loading performance
  • Improving user experience

The author does NOT:

  • Sell visitor data
  • Rent visitor data
  • Exchange visitor data
  • Use visitor data for commercial marketing

7. Information Sharing and Disclosure

This Website will not actively sell or disclose visitor information to third parties, except under the following circumstances:

  • Required by applicable laws or regulations
  • Requested by legal or judicial authorities
  • Necessary to protect the security of the Website
  • Necessary to prevent abuse, attacks, or illegal activities

In such cases, only necessary technical information may be disclosed.

8. External Links

This Website may contain links to external websites, such as:

  • Technical documentation
  • Open-source projects
  • Reference materials
  • Other blogs

These external websites operate independently and have their own privacy policies.

The author is not responsible for:

  • Data collection practices
  • Privacy policies
  • Content security

of third-party websites.

Visitors are encouraged to review the privacy policies of external websites before using them.

9. Data Security

The author makes reasonable efforts to maintain the security of this Website, including:

  • Using reliable hosting platforms
  • Minimizing unnecessary data collection
  • Keeping website components updated
  • Restricting server access

However, due to the open and complex nature of the internet, absolute security of data transmission cannot be guaranteed.

Users should understand and accept the inherent risks associated with using online services.

10. Children’s Privacy

The content of this Website is intended for a general audience and is not specifically directed toward minors.

If minors access this Website, it is recommended that they do so under the supervision of a parent or guardian.

11. User Data Rights
